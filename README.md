MyFamApp is a responsive web application designed to help families stay connected, organized, and informed.  
It allows family members to communicate, share announcements, post updates, and manage tasks in one secure platform.

## Features
- User registration & login
- Family member dashboard
- Shared announcements
- Private messaging between family members
- Photo sharing
- Role-based access (Admin & Members)
- To-do list for family tasks
- Status updates (like stories that expire in 24 hours)
- Inbox for private message exchange

## Technologies Used
- PHP
- MySQL
- HTML
- CSS (Bootstrap)
- JavaScript

## How to Run the Project
1. Install [XAMPP](https://www.apachefriends.org/index.html) or any PHP/MySQL server.
2. Copy the project folder to your `htdocs` directory (or web root).
3. Start **Apache** and **MySQL** via XAMPP control panel.
4. Open phpMyAdmin and import the database file located at `/sql/myfamapp.sql`.
